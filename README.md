# 📊 Operational KPI Dashboard (Power BI)

## 📌 Project Overview
This project presents an Operational KPI Dashboard built using Power BI to monitor SLA compliance, process efficiency, and operational performance.

The dashboard enables stakeholders to identify bottlenecks, track delays, and improve business processes through data-driven insights.

---

## 🎯 Objectives
- Monitor SLA compliance across processes
- Identify delays and inefficiencies
- Track operational KPIs and performance metrics
- Improve visibility into business operations

---

## 🛠 Tools & Technologies
- Power BI
- SQL
- Excel / CSV dataset
- DAX

---

## 📊 Key Features
- SLA Compliance tracking
- Delay analysis by process and region
- KPI cards for performance monitoring
- Interactive filters and slicers
- Top N analysis for bottleneck identification

---

## 📌 Key Insights
- Certain processes consistently exceed SLA targets
- Delays are concentrated in specific regions
- Operational inefficiencies identified through trend analysis
- SLA compliance varies significantly across processes

---

## 🚀 Business Impact
- Reduced reporting errors through validation logic
- Improved operational visibility and performance tracking
- Enabled proactive decision-making and process optimization

---

## 📂 Project Structure
- Dataset: Operational data used for analysis
- Dashboard: Power BI (.pbix file)
